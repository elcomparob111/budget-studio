What Budget Studio stores

Budget Studio is a personal budgeting tool. If you create an account, we store your email, optional display name, and the budget data you enter (categories, amounts, transaction notes) in Supabase (hosted Postgres + Auth). On this device we may also keep a local cache in your browser so the app works offline.

What we do not collect

We do not ask for bank login credentials, account numbers, Social Security numbers, or payment card details. There is no bank aggregation in this app.

How data is protected

Authentication and password hashing are handled by Supabase Auth. Budget rows are protected with Row Level Security so each signed-in user can only access their own data. Encryption at rest and in transit for cloud data is provided by the hosting providers (Supabase / your browser’s HTTPS connection to GitHub Pages).

Your choices

  • Export CSV or JSON backup from Settings.
  • Delete cloud budget data from Settings → Delete cloud data (then sign out).
  • To fully remove your Auth user, delete it in the Supabase project dashboard (Authentication → Users), or contact the app operator.
